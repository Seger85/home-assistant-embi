# Product overview

EMBi is a Home Assistant config-entry integration for Emby players, optional statistics sensors, and safe maintenance of historical server records.

The supported installation source is the latest stable GitHub release consumed through HACS. Runtime configuration is performed through the Home Assistant UI. Credentials and identity-bearing values are redacted from diagnostics.

The integration preserves existing player identities and separates player visibility, sensor lifecycle, entity-registry ownership, server-history cleanup, and persistent scheduler state.
